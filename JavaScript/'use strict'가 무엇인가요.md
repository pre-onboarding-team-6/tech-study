# 'use strict'가 무엇인가요?

## 답변

'use strict'는 파일 전체 혹은 함수 스코프에 strict mode를 사용하게 만드는 명령문입니다.  
strict mode는 안정적인 코드를 생산하기 위해 ES5에 추가된 모드로 오류를 발생시킬 가능성이 높은 코드에 대해 에러를 발생시킵니다. 오류 발생 조건으로는 전역 변수 사용, 매개변수, 객체 속성 이름이 중복, 8진수 사용 등이 있습니다.
